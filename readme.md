# xmonadrc 

## 安装

### vscode 

只需要将预先设定在`.vscode`文件夹下的`task`启动即可(使用`ctrl+shift+b`)

当终端输出`-- set up over`表明编译完成可以使用

### shell

在`shell`环境下运行`.vscode/buildUp.sh`以编译
